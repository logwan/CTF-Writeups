# [Web] Bars
## 問題の概要
- **Difficulty:** Easy
- **Released:** Mar 2, 2026
- **Solved:** 2026-05-25

### 問題文
> Bars, bars, and more bars.

## 方針
指定されたURLへ飛んでみると、クリックもスクロールもキー入力も何もできない。
画面には
```
FLAG: Alpaca{|1||I|l1|IIIl1|1lII|1II|1|I|| ...
```
のみが表示されている。

ダウンロードできるファイルを確認すると、あらゆる入力をブロックするjavascriptがあった。
しかし、このままではFLAGの全文が確認できない。

## 解法
JavaScriptによる制限は、あくまでブラウザのUI上でのみ機能するため、サーバーとの通信そのものを直接確認することにした。
Windows上のUbuntu環境から `curl` コマンドを使用して、生データの取得を試みる。
``` Bash
curl https://[指定URL]
```
これにより、ブラウザの制限を受けずにFLAGの文字列の全貌を取得できた。

-  `curl`コマンドとは
	コマンドラインからHTTPリクエストなどを送信し、サーバーからのデータ（HTMLなど）をそのまま受け取るツール。Webブラウザのように画面を描画したり、JavaScriptを実行したりする機能を持たない。そのため、今回のようなフロントエンド（UI）の制限を完全に無視して、通信の中身だけを安全に覗き見ることができる。

## 結果
コマンドを実行すると、レスポンスのHTMLデータがターミナル上に直接表示された。ブラウザによるUIの制限を受けないため、長いFLAGの全文が確認＆コピーできた。

## 感想
FLAGが長すぎたかつ、意味のありそうな文字列だったため、この後に暗号の復号があるのかと勘違いしてました。これ自身が正解だったことに気が付いてからは、まずはFLAGとして突っ込んでみようと反省しました。
